Внешняя сортировка

Утилита для сортировки файлов большого размера.

Реализованные порядки сортировки:
- лексикографический порядок (возрастание/убывание) - lex
- в порядке увеличения/уменьшения длины - length
- в случайном порядке - random

Формат запуска:
- запуск сортировки в лексикографическом порядке: python3 ext_sorting.py file.txt 256 lex
- запуск сортировки в порядке уменьшения длины: python3 ext_sorting.py file.txt 256 length -r
